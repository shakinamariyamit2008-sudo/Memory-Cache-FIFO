# Memory Cache using FIFO

## Description
This project implements a memory cache system with limited size.

## How it works
When the cache becomes full, the oldest item is automatically removed to make space for a new item.

## Concepts Used
- Queue (FIFO)
- Memory Management
- Array in C

## Example
Cache size = 3  
A → B → C → D  
Result: B C D
